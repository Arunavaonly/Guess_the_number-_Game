Welcome to the 'Guess the Number' Game

Computer has randomly generated a number between 0 to 100. The player's task is to guess the number. A player can attempt maximum 8 times. After each attempt, the player will be informed whether the number is greater or less. If you know how to do it, you will always be able to correctly guess the number within 8 attempts. Are you ready for the challenge? Click on the 'Hint' button if you are clueless. Refresh the page to start again.
